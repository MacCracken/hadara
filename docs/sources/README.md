# Sources

Authoritative references used to verify cultural data in hadara.

Each file corresponds to a data module and lists the primary sources consulted
for dates, classifications, and factual claims. General references that span
multiple modules are in [general.md](general.md).

| File | Module | Coverage |
|------|--------|----------|
| [general.md](general.md) | all | Cultural anthropology, world history |
| [cultures.md](cultures.md) | `culture` | Founding/end dates, geographic origins, status |
| [practices.md](practices.md) | `practice` | Rituals, ceremonies, festivals, daily practices |
| [art-forms.md](art-forms.md) | `art` | Music, dance, visual art, oral tradition |
| [figures.md](figures.md) | `figure` | Key cultural figures, founders, teachers |
| [texts.md](texts.md) | `text` | Sacred/canonical texts, dates, authorship |
| [relations.md](relations.md) | `relation` | Influence, syncretism, exchange, conflict |
